## Structure

gen-np det-case mod-case noun

## Determiners and case marking

| Determiner | Meaning |
|------------|---------|
| ∅ | general |
| shu- | proximal |
| ri- | medial |
| kú- | distal |
| ru- | all |
| ti- | many |
| sä- | some |
| wu- | few |
| u- | none |
| ta- | interrogative |
